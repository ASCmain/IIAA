# IIAA — Intelligent International Accounting Assistant

Assistente IAS/IFRS "grounded" (RAG + guardrail), local-first.

## Struttura
- apps/: CLI tools
- src/: moduli
- data/raw e data/processed: NON versionati
- docs/: policy, blueprint benchmark, appendici tecniche

## Quick start
1) Copiare `.env.example` in `.env` e valorizzare i parametri
2) Seguire il playbook (docs/) e gli script in `apps/`
