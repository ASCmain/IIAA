# Corpus IIAA

Struttura:
- corpus/original/  : documenti originali (PDF, HTML salvati, ecc.). Tipicamente NON versionati.
- corpus/catalog/   : catalogo bibliografico e tecnico (versionato) per citazioni e bibliografia (APA-ready).
- corpus/processed/ : artefatti intermedi (estrazioni testo, chunk, ecc.). Tipicamente NON versionati.

Nota: il catalogo è la sorgente unica di verità per autore/editore/anno, URL e data/ora di accesso.
